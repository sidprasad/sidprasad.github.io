A binary tree is a data structure that consists of nodes, where each node can have at most two children and at most one parent. Exactly one node in the tree has no parent node. This node is called the root node. Every node in the tree is a descendant of this root node. Binary trees are acyclic, meaning that you cannot traverse the tree in a loop. There is exactly one path from the root node to any other node in the tree. A programming student is learning about binary trees. In order to demonstrate their understanding of the data structure, they have built a diagram of a 9 Node binary tree. The edges between the nodes represent parent-child relationships between the nodes.

  • If an edge labelled parent points from NodeA to NodeB, this means that NodeB is the parent of NodeB.
  • If an edge labelled children points from NodeA to NodeB, this means that NodeB is a child of NodeA.
