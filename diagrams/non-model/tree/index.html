A binary tree is a data structure that consists of nodes, where each node can have up to two children (one one the right and one on the left) and at most one parent. A programming student is learning about binary trees. In order to demonstrate their understanding of the data structure, they have built a diagram of a 10 Node binary tree. The edges between the nodes represent parent-child relationships between the nodes.

  • If an edge labelled parent points from NodeA to NodeB, this means that NodeB is the parent of NodeB.
  • If an edge labelled left points from NodeA to NodeB, this means that NodeB is the left child of NodeA.
  • If an edge labelled right points from NodeA to NodeB, this means that NodeB is the right child of NodeA.
A valid binary tree must satisfy the following conditions:
  • Each node in the tree has at most two children. A node can have at most one left child and at most one right child.
  • Exactly one node in the tree has no parent node. This node is called the root node.
  • Every node in the tree is a descendant of the root node.
  • No node in the tree is a descendant of itself.
